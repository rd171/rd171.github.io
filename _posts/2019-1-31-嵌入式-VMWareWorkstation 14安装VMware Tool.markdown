###### VMWareWorkstation 14安装好后一般都自带VMWare Tool，如果不带依次点击如下菜单进行在线更新：  
###### Player -> 帮助 -> 软件更新  
###### 然后进入光驱可以看到VMWare Tool安装包  
###### 将安装包拷贝到Desktop，然后解压
###### 在解压文件夹上右键选择Open terminal，输入如下命令进行安装：  
```
sudo ./vmware-install.pl
```
###### 全部选择y，完成安装
