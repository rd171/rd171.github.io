**ubuntu格式化SD卡脚本** 

1、取得root权限  
sudo -i  

2、查看所有分区  
sudo blkid -o list  

3、然后使用fdisk工具  
fdisk /dev/sdc1  

4、查看SD卡分区  
p  

5、依次删除分区  
d

d  
2  
d

4  

6、查看SD分区  
p  

7、新建分区  
n
