**ubuntu格式化SD卡脚本**

1、取得root权限  
sudo -i  

2、查看磁盘分区详细情况  
sudo fdisk -l  

3、卸载SD卡分区  
sudo umount /dev/sdc1  

4、格式化  
sudo mkfs.vfat /dev/sdc1 -I  

5、 挂载SD卡到sd_card目录  
mount -t vfat /dev/sdc1 /sd_card
