### 1、下载DoSBox
```
https://www.dosbox.com/download.php?main=1  
```
### 2、安装
双击安装包即可进行安装

### 3、打开Dos程序
例如dos程序在D:\testdos\a.exe   
先将Dos程序所在目录映射到DosBox的虚拟C盘
```
mount c d:\testdos
```
运行a.exe
```
c:
a.exe
```
