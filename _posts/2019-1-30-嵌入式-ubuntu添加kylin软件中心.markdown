##### 步骤如下：    
###### 1、桌面空白处右键选择Open terminal  
###### 2、依次输入以下命令:  
```
sudo apt-get update
sudo apt-get install ubuntu-kylin-software-center
```  
###### 3、在任务栏上选择“Search your computer”打开搜索中心  
###### 4、输入kylin，查找已安装的kylin  
###### 5、在任务栏上的kylin图标上点击右键，选择“Lock to Launcher”,以便下次在任务栏上打开
