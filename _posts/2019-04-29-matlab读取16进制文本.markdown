#### 读取代码
```   
clear all;
clc;
src = textread('16_ASII','%s');%将TXT中的数据以文本的方式取回
idx = 1;
for i=1:3:length(src)
    [src{i:i+2}]
    [src{i+2:i+2} src{i+1:i+1} src{i:i}]
    dest1(idx) = hex2dec([src{i:i+2}]);
    dest(idx) = hex2dec([src{i+2:i+2} src{i+1:i+1} src{i:i}]);
    idx = idx + 1;
end
x=1:length(dest);
plot(x,dest,'r', x, dest1, 'b');
```
