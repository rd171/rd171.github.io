#### 构造一个像素的虚线画笔  
```
CPen m_penGrid;
m_corGrid = RGB(255, 0, 255);
LOGBRUSH LogBrush;
LogBrush.lbColor = m_corGrid;
LogBrush.lbStyle = PS_SOLID;
m_penGrid.CreatePen( PS_COSMETIC | PS_ALTERNATE , 1, &LogBrush, 0, NULL );
```
