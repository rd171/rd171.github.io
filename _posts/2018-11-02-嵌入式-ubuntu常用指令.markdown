##### 查看当前路径  
```
pwd
```
##### 查看当前路径文件  
```
ls  
```
##### 新建文件夹abc  
```
mkdir /abc
```

##### 删除文件夹abc
```
rm -rf /abc
```

##### 查看网络  
```
ifconfig  
```

##### 查看网络设备配置  
```
cd wifi
ls
cat ipaddr
cat lan_addr
```

##### 查看文件abc
```
cat abc
```

##### 查看所有进程
```
ps
```  

##### 查看内存、CPU和进程情况
```
top
```  

##### 结束进程
```
kill 9 PID
```

##### 修改IP地址
```
sudo ifconfig eth0 192.168.1.100 broadcast 192.168.1.255 netmask 255.255.255.0
```

##### 修改网关
```
sudo route add default gw 192.168.1.1 dev eth0
```

##### 修改DNS
```
编辑/etc/resolv.conf文件，增加
nameserver 202.106.0.20
```
