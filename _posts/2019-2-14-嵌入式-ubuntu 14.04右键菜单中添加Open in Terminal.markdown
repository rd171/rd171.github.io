###### ubuntu 14.04右键菜单中添加Open in Terminal      
###### 1.打开一个Terminal（ctrl+alt+t），输入如下指令  
```
sudo apt-get install nautilus-open-terminal
```    
###### 2.使用以下指令来重启Nautilus  
```
nautilus -q
```
