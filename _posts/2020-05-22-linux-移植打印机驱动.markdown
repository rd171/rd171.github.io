### 1、编译CUPS
#### 1.1、下载CUPS源码
```
https://github.com/apple/cups/releases/tag/v2.3.3
```
#### 1.2、进入CUPS源码目录
```
cd ~/Downloads/cups/cups-2.3.3
```
#### 1.3、配置环境
```
./configure --host=arm-linux --prefix=/opt/cups
```
#### 1.4、编译
```
make
```
#### 1.5、安装
```
sudo make install
```
opt/cups目录即为cups程序文件
